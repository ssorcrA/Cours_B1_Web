Mémento HTML5

Les balises de titre

<h1> à <h6> : Permettent de structurer les titres.

Exemple :

<h1>Titre principal</h1>
<h2>Sous-titre</h2>

Les balises de texte

<p> : Pour les paragraphes.

Exemple :

<p>Ceci est un paragraphe.</p>

Les balises de texte en gars

<strong> : Pour les paragraphes.

Exemple :

<strong>Ceci est un paragraphe.</strong>

Les balises de texte pour surlignée

<mark> : Pour les paragraphes.

Exemple :

<mark>Ceci est un paragraphe.</mark>

Les balises de retour à la ligne

<br> : Pour les paragraphes.

Exemple :

<br>Ceci est un paragraphe.</br>

Les balises de points

<li> : Pour les paragraphes.

Exemple :

<li>Ceci est un paragraphe.</li>

Les balises de soulignage

<u> : Pour les paragraphes.

Exemple :

<u>Ceci est un paragraphe.</u>

Les balises de boutton

<button onclick="window.location.href=> : Pour les paragraphes.

Exemple :

<button onclick="window.location.href=>Ceci est un paragraphe.</button>

Les balises de lien

<< a href="adresse de redirection" >> : Pour les paragraphes.

Exemple :

<< a href="adresse de redirection" >>Ceci est un paragraphe.</a>

Les balises d'image

< img src="URL source" > : Pour les paragraphes.

Exemple :

< img src="URL source" >Ceci est un paragraphe.</img>
